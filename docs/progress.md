## 已确认产品决策

- 首次上传后建议开启定时采集，但默认关闭且必须由用户主动同意；支持每 6 小时、每日和仅手动。
- 大部分账号为个人订阅。无法识别真实账单时显示各模型相加的 API 等价总价并标 `≈`；用户可选填订阅月费，单独展示订阅支出。
